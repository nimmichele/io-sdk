IO SDK

Thank you for installing IO-SDK.

Initializing IO-SDK

For the first time, open a new terminal and run

  $ iosdk init
Starting IO-SDK

From a terminal run

  $ iosdk start

Uninstall IO-SDK

Run the following command to uninstall IO_SDK.

  $ sudo bash /Library/__PRODUCT__/__VERSION__/uninstall.sh
